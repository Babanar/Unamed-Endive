Titre : # blabla
     \_____> englobe souvent une scène

Personne(s) présente(s) dans la scène : [personne1, ...]

Description des images : -description-
                     \____> image qu'il faudra bien entendu afficher

Note du créateur : (*note: contenu ...*)

Description d'une cinématique : *description*

Texte : -> blabla
   ou : nom_de_la_personne : blabla
   Nota bene : très très souvent (genre tout le temps), "->" est utilisé pour introduire les paroles du prof
   Nota bene bis : System: texte affichera un texte dans une bulle bleue

Entrée utilisateur demandée : INPUT variable_où_stocker_l'input

Choix utilisateur demandé : CHOICE texte [choix1, ...] variable_où_stocker_le_choix

Conditions : IF [condition] THEN
            ELIF [condition] THEN
            ELSE THEN
            EndIF

Quand un event se passe : OnEVENT [event] DO ... EndEVENT

Variable : {nom_de_la_variable}

Notes :  - régions fictives souvent évoquées dans l'histoire : - Tanjoh
                                                               - Ult'af
        - liste des variables utilisées : - player_name
                                          - tu_ecoutes
